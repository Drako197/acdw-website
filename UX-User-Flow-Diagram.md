## User Flow Decision Tree

```
HOMEPAGE ENTRY
    │
    ├── HERO SECTION
    │   ├── "Buy Now" → Mini Product Page → Checkout
    │   └── "See How It Works" → Demo Request → Contact Form
    │
    ├── PRODUCT SHOWCASE
    │   ├── Sensor Card
    │   │   ├── "Learn More" → Product Details → Purchase
    │   │   └── "Buy Now" → External Product Page
    │   │
    │   ├── Core 1.0 Card
    │   │   ├── "Learn More" → Product Details → Purchase
    │   │   └── "Buy Now" → Product Page → Checkout
    │   │
    │   └── Combined Card
    │       ├── "Learn About Bundle" → Bundle Details → Purchase
    │       └── "Contact HVAC Pro" → Contact Form → Consultation
    │
    └── CUSTOMER TYPE SELECTION
        ├── Homeowners → Direct Purchase Flow
        ├── HVAC Professionals → Sign-in → Professional Pricing
        ├── Distributors → Sign-in → Distributor Pricing
        └── Code Officials → Sign-in → Compliance Information
```
